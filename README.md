## WMC Capital Group – Website Redesign

Next.js + Tailwind CSS implementation of the new WMC Capital Group marketing site, inspired by the clean, investment‑focused presentation of `theschaffgroup.com`.

### Tech stack

- **Framework**: Next.js (App Router, TypeScript)
- **Styling**: Tailwind CSS with a custom global design system

### Running the project

```bash
npm install
npm run dev
```

Then open `http://localhost:3000` in your browser.
